# 🚀 Web App & Game Playground

個人的に作成したツールやミニゲームをまとめたリポジトリです。
パブリックにしているので、友達のみんなはここからコードを落としたり、ブラウザで開いたりして自由に使ってください！

## 🎮 すぐに遊べる・使えるもの

### 1. P2P対戦ゲーム（ブラウザで開くだけ！）

サーバー不要で、友達と直接通信して遊べるゲームです。

* **[Air Hockey](https://www.google.com/search?q=air_hockey_p2p.html) / [Pong**](https://www.google.com/search?q=pong_p2p.html)
* **遊び方:** 片方が自分のIDを伝えて、もう片方がそれを入力して「接続」するだけ！

### 2. Auto Battle RPG

Reactで作った、眺めて楽しむタイプのRPGです。

* **[RPGを開く](https://www.google.com/search?q=minigame.html)**

### 3. AWSサーバー管理ツール (PHP)

MinecraftやPalworldのサーバーをブラウザからポチッと起動・停止するためのツールです。

* **ファイル:** `start_server.php`
* **注意:** これを使うには自分のサーバーに設置して、AWSの設定をする必要があります。

---

## 🛠 セットアップ（友達向け）

### AWSツールを使いたい場合

1. `config.php.example` を `config.php` にリネームします。
2. 中身に自分のAWSの鍵（Access Keyなど）を書き込みます。
3. **【超重要】** `config.php` は絶対にGitHubにアップ（Push）しないでください！
4. 誰でも操作できないように、サーバー側でパスワード（Basic認証）をかけることを強くおすすめします。
